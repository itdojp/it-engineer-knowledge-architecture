# Professional Foundations テンプレート集

このディレクトリは、基礎リテラシー（Professional Foundations）カテゴリで横断的に使う最小テンプレートをまとめたものです。
各書籍の詳細テンプレートへ進む前に、OJT、研修、社内標準化の初期セットとして利用できます。

## テンプレート一覧

- `README-template.md`: プロジェクト概要・利用手順・責任範囲
- `issue-template.md`: Issue / タスク / 調査依頼
- `pr-template.md`: Pull Request 説明
- `procedure-template.md`: 手順書
- `runbook-template.md`: Runbook
- `adr-template.md`: Architecture Decision Record
- `incident-report-template.md`: 障害報告 / ポストモーテム
- `progress-report-template.md`: 進捗報告

## 運用ルール

1. `<...>` のプレースホルダーを具体値に置き換える。
2. 分からない項目は削除せず、「未確認」「該当なし」「判断待ち」と明記する。
3. 秘密情報、個人情報、顧客固有情報は貼り付けず、保管場所・参照権限・マスキング方針を記載する。
4. PR では、変更したテンプレートと利用想定をレビュー観点に含める。

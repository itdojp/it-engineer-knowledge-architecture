# 著作権・ライセンス表示ガイドライン

## 目的

読者に対して「非営利利用は可能」「商用利用は別契約」という条件を誤解なく伝えることを目的とします。

## 推奨表示箇所

- リポジトリルート: `LICENSE.md`（正式な条件）
- 書籍トップページ/README: 簡易説明（本文=CC BY-NC-SA 4.0、商用は別契約、窓口）
- サイトフッター: 短い表記（例: CC BY-NC-SA 4.0 / commercial use requires agreement）

# 新規書籍クイックスタート

本ページは、新規書籍を立ち上げる際に最低限確認する項目を、既存ガイドから抜粋して並べた導入用チェックリストです。

## 最短手順

1. `templates/book/` を起点に、標準ディレクトリ構成と必要ファイルを確認する。
2. `book-config.json` で `ux.profile` と `ux.modules` を決める。
3. ライセンス表記と `CHANGELOG.md` の初期内容を整える。
4. 原稿を追加したら、公開前にレビュー・チェックリストを一通り確認する。

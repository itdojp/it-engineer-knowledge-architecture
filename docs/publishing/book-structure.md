# 各巻の標準構成

各書籍リポジトリは、読者の可読性と更新運用の一貫性を確保するため、以下の構成を推奨します。

1. 扉（タイトル/版/著者/公開URL等）
2. 著作権・ライセンス（本文=CC BY-NC-SA 4.0、商用は別契約）
3. まえがき（目的/想定読者/前提/検証環境/読み方/フィードバック窓口）
4. 目次（自動生成または固定）
5. 本文
6. あとがき（まとめ/次のステップ/運用上の注意/フィードバック）
7. 更新履歴（版/日付/変更概要）
8. 奥付（連絡先/初版/現行版等）

テンプレートは `templates/book/` を参照してください。
